Pergunta 11: O que é aninhamento?

O Aninhamento nos permite criar estruturas que contenham outras estruturas internas
como por exemplo:

var nome = 'Raffael'
var nome1 = 'raffael'

if (nome === nome1){
    console.log('Nomes iguais')
}else {
    console.log('Nomes diferentes')
}

Essa estrutura do if, é um exemplo de estrutura aninhada
onde o as instruções estão "dentro" (aninhadas) do if.
