Pergunta 17: Qual a relação entre nulo e indefinido ?

Existe uma diferença semântica entre null e undefined: o valor null é utilizado para indicar a ausência de um objeto,
enquanto undefined indica a ausência de um valor qualquer.
A especificação explicita isso quando define os valores undefined e null:

Valor undefined

Valor primitivo utilizado quando uma variável não teve valor atribuído.

Valor null

Valor primitivo que representa a ausência intencional de um valor de objeto.
