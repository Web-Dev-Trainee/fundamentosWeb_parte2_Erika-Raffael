Pergunta 15: Qual a sintaxe para uma instrução switch?

A condicional switch avalia uma expressão, combinando o valor da expressão para um cláusula case, e executa as instruções associadas ao case
exemplo da sintaxe:

switch (expressão) {
  case valor1:
   //Instruções executadas quando o resultado da expressão for igual á valor1
    break;
   case valor2:
   //Instruções executadas quando o resultado da expressão for igual á valor2
    break;
     ...
   default:
   //Instruções executadas quando o valor da expressão é diferente de todos os cases
    break;
}

Exemplo de Switch:

const expr = 'Banana';
switch (expr) {
  case 'Laranja':
     console.log('As Laranjas estão custando R$4,50.');
    break;
  case 'Morango':
  case 'Banana':
     console.log('Os Morangos e Bananas estão custando R$12,70.');
     break;
  default:
     console.log(`Desculpe, mas estamos sem ${expr}.`);
}
