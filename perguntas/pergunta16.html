Pergunta 16: Qual a sintaxe para um operador ternário?

O operador ternário é o único operador JavaScript que possui três operandos. Este operador é frequentemente usado como um atalho para a instrução if.

Exemplo de sintaxe:

condição ? expr1 : expr2

condição:
   Uma expressão que é avaliada como true ou false.

expr1, expr2:
  Expressões com valores de qualquer tipo.

Exemplo de uma operação ternária:

var ehMembro = false;
var taxa = ehMembro ? "R$2.00" : "R$10.00";
console.log(taxa);
