Pergunta 14: Qual a sintaxe para uma condicional if e else?

A sintaxe usada para essa condicional é escrita da seguinte forma:

if (condição a se verificada){
    retorno se for comprida a condição
}else{
     retorno se não for comprida a condição
}

Um exemplo de uma condicional if e else:

a = 5
b = 3
c = a + b
if (c == 8){
    console.log('Condição comprida')
}else{
    console.log('Condição não comprida')
}
